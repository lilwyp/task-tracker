Задача: Создать одностраничное веб-приложение "Трекер задач".

Описание:
- Это приложение позволяет пользователям добавлять, удалять и отмечать задачи как выполненные.
- Все данные будут храниться в браузере с использованием `localStorage`, так что при обновлении страницы данные не потеряются.

Требования:
1. HTML:
   - Создай простую структуру страницы с заголовком, полем для ввода задачи и кнопкой "Добавить".
   - Добавь список задач (изначально пустой), который будет заполняться новыми задачами.

2. CSS:
   - Сделай приложение приятным на вид: выровняй элементы, используй приятные цвета и шрифты.
   - Добавь интерактивные стили: например, меняй цвет выполненной задачи.

3. JavaScript:
   - Добавь функционал для добавления новой задачи в список.
   - Реализуй возможность удаления задачи и отметки задачи как выполненной.
   - Все данные должны сохраняться в `localStorage`, чтобы при перезагрузке страницы задачи оставались.

4. Дополнительное задание (опционально):
   - Реализуй возможность редактирования задач.
   - Добавь фильтр для показа только выполненных, невыполненных или всех задач.

Этот проект поможет тебе показать владение базовыми навыками HTML, CSS и JavaScript. Удачи!
